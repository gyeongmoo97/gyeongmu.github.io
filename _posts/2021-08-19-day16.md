## Flex Responsive (반응형)

```css
.flex-container {
  display: flex;
  flex-direction: row;
}

/* Responsive layout - makes a one column layout instead of a two-column layout */
@media (max-width: 800px) {
  .flex-container {
    flex-direction: column;
  }
}

```

기기의 폭이 800px 이하면 아래의 속성을 적용한다는 내용.
